## Description

HTML Editor

## 구현 기능

- 범위에 폰트 스타일을 적용 할 수 있는 기능 개발 함.
- 범위에 폰트 스타일을 적용 할 때, 스타일이 겹치면 반드시 적용 할 수 있게, 하위 내용을 삭제 할 수 있는 기능 개발 함.
- 작성한 히스토리를 관리하며 undo redo 기능을 개발 함.
- image 삽입 및 리사이즈 기능 개발 함.

## 사용한 API

- selection API, RangeAPI

## 기술 스택

- typescript
- webpack
